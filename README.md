# AFK-bot
Grand RP AFK Bot
Installation & Usage
Step 1 — Download

Download AFK.exe from the latest release.

Step 2 — Prepare Grand RP
Launch Grand RP.
Enter your own house or a rented house.
Do not use a family house
Step 3 — Change Display Mode

Open the game settings and set the Screen Mode to Windowed.

Step 4 — Pin Grand RP to the Taskbar

Pin Grand RP to your Windows taskbar and make sure it is the first pinned application, like this:
<img width="318" height="44" alt="Screenshot 2026-07-14 145156" src="https://github.com/user-attachments/assets/1fd869de-8d4e-4aae-8301-a9e06454e9a6" />


Step 5 — Test the Shortcut

Before using the bot, verify that the shortcut works:

Press Windows + 1 → Grand RP should open.
Press Windows + 1 again → Grand RP should minimize.

If this doesn't work, move Grand RP to the first position on your taskbar and test again.

Step 6 — Start the Bot
Run AFK.exe.
Click Start.

Important: Only start the bot while your character is inside your own house or a rented house.

Notes
This project is intended for educational and experimental purposes.
If you have any concerns about the executable, feel free to scan it with your preferred antivirus or an online service such as VirusTotal before running it.
The bot is designed for Windowed mode only.
