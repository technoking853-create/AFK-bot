#  Grand RP AFK Bot

A lightweight AFK automation tool for **Grand RP**.

> ⚠️ **Educational & Experimental Project**

---

## 📥 Installation

1. Download **`AFK.exe`** from the **Releases** page.
2. Place it anywhere on your PC.

---

## Setup

### 1. Enter a Personal Property

Launch **Grand RP** and enter:

- ✅ Your own house
- ✅ A rented house
- ❌ Family houses are **not supported**

---

###2. Enable Windowed Mode

Open **Settings → Display**

Set:

```
Screen Mode → Windowed
```

---

### 📌 3. Pin Grand RP to the Taskbar

Pin **Grand RP** to your Windows taskbar and move it to the **first position**.

Example:
<img width="318" height="44" alt="Screenshot 2026-07-14 145156" src="https://github.com/user-attachments/assets/a891f018-4cb1-4870-9759-cc0fb884086c" />

---

### ⌨️ 4. Test the Shortcut

Press:

```
Windows + 1
```

Expected result:

- ✅ Grand RP opens
- ✅ Press **Windows + 1** again and it minimizes

If this doesn't work, Grand RP is **not** in the first taskbar position.

---

### 5. Start the Bot

1. Launch **AFK.exe**
2. Click **Start**

> 💡 Only start the bot while your character is inside a supported property.
